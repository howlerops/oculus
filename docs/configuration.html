Configuration

Config Directory

Default: ~/.oculus/. Override with OCULUS_CONFIG_DIR env var.

Migrates automatically from ~/.claude/ on first run.

OCULUS.md

Project-specific instructions loaded into every system prompt. Place at project root or .oculus/OCULUS.md.

Also reads CLAUDE.md as fallback for compatibility.

Supports @include path/to/file.md directives.

Environment Variables

VariableDescription
ANTHROPIC_API_KEYAPI key (skips OAuth login)
OCULUS_CONFIG_DIRCustom config directory
OPENAI_API_KEYFor OpenAI bridge provider
